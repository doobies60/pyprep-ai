Jinja2 テンプレート継承:

親ファイル: base.html (共通レイアウト)

子ファイル: index.html, login.html 等

メリット:

デザイン変更時に base.html 1か所を直せば全画面に反映される。

各HTMLファイルが短くなり、管理しやすくなる。
